# Acknowledgments

## MobileCoin Foundation

Botho is substantially derived from the [MobileCoin](https://github.com/mobilecoinfoundation/mobilecoin) project, originally developed by MobileCoin Inc. and the MobileCoin Foundation.

We gratefully acknowledge the MobileCoin team for their foundational work on privacy-preserving cryptocurrency technology, including:

- Secure enclave-based transaction validation
- Fog service for mobile-friendly private transactions
- Implementation of the CryptoNote protocol
- Stellar Consensus Protocol (SCP) implementation

The original MobileCoin code is licensed under a combination of Apache 2.0 (for client SDK components) and GPL v3 (for server-side components). Please refer to the LICENSE file for complete licensing details.

**Original Repository:** https://github.com/mobilecoinfoundation/mobilecoin

We are committed to maintaining proper attribution and complying with all applicable open source licenses as we develop Botho.
