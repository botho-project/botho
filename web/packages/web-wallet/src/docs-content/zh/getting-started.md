## Botho 快速开始

Botho 是一种以隐私为核心的加密货币，专为后量子时代设计。它将用于交易隐私的**隐匿地址**与用于快速、节能共识的**恒星共识协议（SCP）**相结合。与工作量证明加密货币不同，Botho 在保持强隐私保证的同时，能够在数秒内达成最终性。

### Botho 有何不同？

像比特币这样的传统加密货币拥有透明的区块链，任何人都可以追踪地址之间的资金流动。即便是所谓的“隐私币”，也常常依赖于可能被未来量子计算机攻破的密码学假设。

Botho 采取了不同的方法：

- **隐匿地址**确保你收到的每一笔付款都发往一个唯一的一次性地址，使得任何人都无法通过观察区块链将你的交易关联起来
- **后量子密码学**保护你的隐私，抵御拥有量子计算机的对手
- **联邦拜占庭协议**提供快速最终性——共识安全从不依赖算力
- **平等主义发行**通过 RandomX CPU 挖矿来分发新币，且刻意与共识解耦：挖矿赚取奖励，但不会因此获得对哪些交易被确认的话语权
- **渐进式经济机制**——手续费随财富集中程度在 1×–6× 之间浮动，每笔手续费的 80% 通过抽奖重新分配，20% 被销毁

### 创建钱包

开始使用 Botho 只需几个步骤：

1. **访问钱包页面** - 从主页点击“打开钱包”，或直接导航至钱包
2. **选择“创建新钱包”** - 如果你有恢复助记词，也可以导入现有钱包
3. **保护你的恢复助记词** - 你将看到一组 24 个单词的助记词。请将其写在纸上并存放在安全的地方。如果你失去了对设备的访问权限，这组助记词是恢复资金的**唯一方式**
4. **可选：设置密码** - 添加一个加密密码以增强安全性。每次在此浏览器中打开钱包时都需要输入该密码

**重要提示：** 切勿与任何人分享你的恢复助记词。任何拥有这些单词的人都能访问你的资金。切勿以数字形式存储它（不要截图，不要云存储，不要密码管理器）。

### 理解你的钱包地址

你的钱包地址形如：`botho://1/4nuKn2U5qsRk3vD...`（总共约 90 个字符）

该地址格式包含：
- **协议标识符**（主网为 `botho://`，测试网为 `tbotho://`）- 不同的前缀可防止意外的跨网络发送
- **地址版本**（`1/`，即当前地址版本）
- **公钥** - 你的查看密钥和花费密钥，一起以 base58 编码

不存在单独的抗量子地址类型：早先的 `1q/` 量子地址层级已被移除（ADR 0006）。收款人的后量子隐私在设计上是普遍适用的——ML-KEM 密钥派生自你的标准地址，因此每个地址都保持相同的简短格式。

你可以放心地将此地址分享给任何想要向你发送资金的人。得益于隐匿地址，每一笔进账交易都会发往一个唯一的派生地址，只有你才能从中花费。

### 接收你的第一笔付款

当有人向你发送 BTH 时：

1. 他们使用你的公开地址派生出一个唯一的一次性地址
2. 交易被广播到网络并被打包进一个区块
3. 你的钱包扫描新区块并检测发给你的付款
4. 资金出现在你的余额中，通常在一个区块内——出块时间会随网络负载自适应，从流量极高时的 3 秒到网络空闲时的 40 秒不等

### 发送付款

要向他人发送 BTH：

1. 点击钱包中的**发送**按钮
2. 输入收款人的 Botho 地址
3. 输入要发送的金额
4. 查看交易详情，包括手续费
5. 确认交易

交易一经确认即为最终——Botho 中没有退单或撤销。

### 交易手续费

每笔 Botho 交易都需要一小笔手续费。这些手续费有三个用途：

1. **防止垃圾信息** - 手续费使得用垃圾交易淹没网络的成本高昂
2. **渐进式征税** - 手续费根据集群财富在 1× 到 6× 之间浮动，在不引发女巫攻击的前提下抑制财富集中
3. **重新分配与通缩** - 每笔手续费的 80% 通过抽奖重新分配给持有者；其余 20% 被永久销毁

手续费基于大小，而非金额：`fee = per-byte rate × transaction size × cluster factor (1×–6×) × output penalty`。详情请参阅“集群标签”和“代币经济”章节。

### 安全最佳实践

- **备份你的恢复助记词**，写在纸上，存放在安全的地方
- **使用密码**在浏览器中加密你的钱包
- **考虑运行你自己的节点**以获得最大隐私
- **发送资金前仔细核对地址**——交易无法撤销
